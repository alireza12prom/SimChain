# SimChain

A fun, simple and educational blockchain project, implemented by Golang, just to get familiar with blockchain stuff.

## Tech Stack

- Golang `v1.25.0`
- Gin `v4`
- BadgerDB `v4`

## Directory Structure

```
- internal
  - core
    - block.go
    - blockchain.go
    - transaction.go
  - main.go (start point)
```
